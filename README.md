# Parcial de Programación Web

Un proyecto simple de página web estática que muestra productos, permite agregar al carrito, cambiar a modo oscuro y validar un formulario de contacto con JavaScript.

##Estructura del proyecto

- `index.html` — Página principal con contenido y formulario.
- `css/styles.css` — Estilos básicos de la página.
- `js/script.js` — Lógica de interacción (cambio de texto, modo oscuro, carrito, validación de formulario).
- `assets/` — Imágenes de productos.

## Cómo ejecutar

1. Abre `index.html` en tu navegador.
2. Navega por los productos y prueba las interacciones:
   - Botón para ver ofertas.
   - Modo oscuro.
   - Agregar productos al carrito.
   - Enviar el formulario de contacto.

## Funcionalidades incluidas

- Cambio de texto dinámico en la página.
- Modo oscuro aplicando estilos al `body`.
- Contador de productos agregados al carrito.
- Validación básica de formulario con alertas.

## Notas

- El proyecto está pensado como un ejercicio académico.
- La validación del formulario se realiza en el cliente y no envía datos a un servidor.
